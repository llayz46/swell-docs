## Installer Swell

Vous pouvez installer **Swell** de deux manières : soit avec l’outil Laravel Installer, soit directement via Composer.

```bash tab="Laravel Installer"
laravel new nom-du-projet --using=swell/starter-kit
```

```bash tab="Composer"
composer create-project swell/starter-kit nom-du-projet
```

Ces commandes vont :
- Télécharger la dernière version de Laravel
- Installer le starter-kit Swell et ses dépendances
- Préconfigurer le projet pour un usage e-commerce moderne

## Configuration de l’environnement

1. Copiez le fichier d’exemple d’environnement :
   ```bash
   cp .env.example .env
   ```
2. Générez la clé d’application :
   ```bash
   php artisan key:generate
   ```
3. Configurez la connexion à votre base de données dans `.env`.
    <Callout>
       Pour toute question concernant la configuration de services comme Algolia ou Stripe, référez-vous à la documentation officielle de [Laravel](https://laravel.com/docs).
   </Callout>

### Modules optionnels

**Swell** propose plusieurs modules optionnels que vous pouvez activer selon vos besoins. Configurez-les dans votre fichier `.env` :

```bash
# Modules optionnels
SWELL_WISHLIST_ENABLED=false
SWELL_BANNER_ENABLED=false
SWELL_REVIEW_ENABLED=false
SWELL_LOYALTY_ENABLED=false

# Configuration du système de fidélité (si activé)
SWELL_LOYALTY_POINTS_PER_EURO=10
SWELL_LOYALTY_EXPIRATION_DAYS=365
SWELL_LOYALTY_MIN_REDEEM=100
SWELL_LOYALTY_MAX_DISCOUNT=50

# Configuration des commandes
SWELL_ORDER_PREFIX="ORD"
```

Pour plus d'informations sur les modules, consultez la page [Gestion des modules](/modules-management).

## Installer les dépendances front-end

```bash tab="npm"
npm install
```

```bash tab="pnpm"
pnpm install
```

```bash tab="yarn"
yarn install
```

```bash tab="bun"
bun install
```

## Initialiser la base de données

Lancez les migrations et seeders de base
pour l'environnement de développement avec des données de test :

```bash
php artisan migrate

php artisan db:seed --class=RoleSeeder

php artisan db:seed --class=LocalSeeder
```

Ou bien, si vous souhaitez, vous pouvez utiliser la commande suivante :

```bash
php artisan swell:install
```

## Lancer le projet en local

Pour démarrer le serveur de développement :

```bash
composer run dev
```

Cette commande lance :
- Le serveur Laravel
- Le watcher Vite pour le front-end
- Le listener de queue
- L’écoute des webhooks Stripe (si configuré)

Accédez ensuite à [http://localhost:8000](http://localhost:8000).

## Se connecter à l’application

Après l’installation, le LocalSeeder créera automatiquement un utilisateur administrateur.  
Les identifiants par défaut sont :

- **Email** : john@doe.com
- **Mot de passe** : password

<Callout>
    Vous pouvez modifier les informations du compte directement dans le `LocalSeeder`. Voir la documentation sur les [seeders](/seeders).
</Callout>
