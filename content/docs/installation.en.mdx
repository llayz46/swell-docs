## Installing Swell

You can install **Swell** in two ways: either with the Laravel Installer tool, or directly via Composer.

```bash tab="Laravel Installer"
laravel new project-name --using=swell/starter-kit
```

```bash tab="Composer"
composer create-project swell/starter-kit project-name
```

These commands will:
- Download the latest version of Laravel
- Install the Swell starter-kit and its dependencies
- Pre-configure the project for modern e-commerce usage

## Environment configuration

1. Copy the example environment file:
   ```bash
   cp .env.example .env
   ```
2. Generate the application key:
   ```bash
   php artisan key:generate
   ```
3. Configure your database connection in `.env`.
    <Callout>
       For any questions regarding service configuration like Algolia or Stripe, refer to the official [Laravel](https://laravel.com/docs) documentation.
   </Callout>

### Optional modules

**Swell** offers several optional modules that you can enable according to your needs. Configure them in your `.env` file:

```bash
# Optional modules
SWELL_WISHLIST_ENABLED=false
SWELL_BANNER_ENABLED=false
SWELL_REVIEW_ENABLED=false
SWELL_LOYALTY_ENABLED=false

# Loyalty system configuration (if enabled)
SWELL_LOYALTY_POINTS_PER_EURO=10
SWELL_LOYALTY_EXPIRATION_DAYS=365
SWELL_LOYALTY_MIN_REDEEM=100
SWELL_LOYALTY_MAX_DISCOUNT=50

# Order configuration
SWELL_ORDER_PREFIX="ORD"
```

For more information about modules, see the [Module management](/modules-management) page.

## Install front-end dependencies

```bash tab="npm"
npm install
```

```bash tab="pnpm"
pnpm install
```

```bash tab="yarn"
yarn install
```

```bash tab="bun"
bun install
```

## Initialize the database

Run the base migrations and seeders
for the development environment with test data:

```bash
php artisan migrate

php artisan db:seed --class=RoleSeeder

php artisan db:seed --class=ProdSeeder
```

Or, if you prefer, you can use the following command:

```bash
php artisan swell:install
```

## Launch the project locally

To start the development server:

```bash
composer run dev
```

This command launches:
- The Laravel server
- The Vite watcher for the front-end
- The queue listener
- Stripe webhook listener (if configured)

Then access [http://localhost:8000](http://localhost:8000).
