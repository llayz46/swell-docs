## Système d'avis produits

**Swell** propose un système d'avis et d'évaluations produits en tant que **module optionnel** permettant aux clients de partager leurs expériences et d'aider d'autres utilisateurs dans leurs décisions d'achat.

<Callout type="info">
  Le système d'avis est un module optionnel. Consultez la page [Gestion des
  Modules](/modules-management) pour l'activer dans votre installation.
</Callout>

## Architecture du système

Le système d'avis est organisé sous forme de module dans l'architecture **Swell**. Toute la fonctionnalité est centralisée dans le dossier `app/Modules/Review/` qui contient :

- **database/** - Migrations et factories spécifiques au module
- **Http/** - Contrôleurs, requêtes et ressources API
- **Models/** - Modèles Review

Cette organisation modulaire permet une activation/désactivation simple via la [gestion des modules](/modules-management) et facilite la maintenance du code.

## Interface utilisateur React

### Composants d'avis

**ReviewForm** - Formulaire de soumission d'avis

- Système d'étoiles interactif pour la notation
- Champs titre et commentaire
- Validation côté client

**ReviewSection** - Composant globale regroupant le formulaire et les avis

- Ajout d'un avis uniquement si l'utilisateur a acheté le produit
- Affichage de la moyenne globale
- Listing de quelques avis

**Review** - Affichage d'un avis

## Configuration du module

<Callout type="warning">
  Avant d'utiliser le système d'avis, assurez-vous qu'il est activé dans votre
  configuration. Voir [Gestion des Modules](/modules-management#reviews-module).
</Callout>

## Routes

```php title="routes/web.php"
Route::middleware('feature:review')->prefix('reviews')->group(function () {
    Route::post('/', [ReviewController::class, 'store'])->name('reviews.store');
    Route::put('/{Review}', [ReviewController::class, 'update'])->name('reviews.update');
});
```

## Fonctionnalités avancées

- **Unicité** - Un seul avis par utilisateur par produit
- **Statistiques** - Calculs automatiques des moyennes et distributions
- **Pagination** - Gestion des grandes listes d'avis

<Callout>
  Le système d'avis améliore la confiance des clients et peut augmenter les
  conversions. Pour plus d'informations sur l'activation, consultez la
  [documentation des modules](/modules-management).
</Callout>

Le système d'avis offre une solution complète pour collecter et afficher les retours clients de manière professionnelle.
